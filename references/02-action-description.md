# 动作描写技巧

## 常见问题

- 动作笼统模糊（"他走到桌边"）
- 缺少内心层次
- 所有年龄段角色动作一样（老人和少年走路没区别）
- 一连串动作压缩成一瞬间

## 动作拆解法

把一个大的笼统动作拆成3-5个小动作，像慢镜头一样逐帧描写。

**写法对比**：
- 笼统式：林晚走到桌边，拿起那封信看了一眼。
- 拆解式：林晚从门口走到桌前，手指在桌面上轻轻滑过。他低头扫了一眼信封上的署名，指尖顿了顿。拿起信来，他没有急着拆，而是先翻到背面看了看封口——蜡封完好。这才用拇指挑开信封口。

**要点**：
- 每个小动作之间插入细微的观察/判断/犹豫
- 动作越关键（重要剧情节点），拆得越细
- 日常动作可以保持简洁，避免节奏拖沓

## 次级动作法

用身体语言（次级动作）代替内心独白，让读者"看到"角色的情绪，而不是"读到"。

**写法对比**：
- 内心独白式：苏念把玉佩还给对方，心里很难过，但她不想表现出来。
- 次级动作式：苏念把玉佩放在桌上，指尖按着玉佩上的穗子，按了很久才松开。站起身时她没有看他，只是盯着桌面上的木纹说："那我走了。"

**常见次级动作清单**：

| 情绪 | 次级动作 |
|------|---------|
| 紧张 | 攥衣角、咬嘴唇、抖腿、反复拨弄手指 |
| 犹豫 | 手指在某个物件上反复摩挲、欲言又止 |
| 愤怒 | 握拳、太阳穴跳动、呼吸变粗、脖颈青筋 |
| 悲伤 | 盯着某处发呆、机械性地重复某个动作 |
| 心虚 | 不敢直视、眼神飘忽、频繁吞咽 |
| 兴奋 | 来回踱步、语速加快、手势增多 |

**原则**：能用一个动作"演"出来的情绪，绝不多写一句独白。

## 年龄特征动作

不同年龄段的角色，做同一件事时动作特征完全不同：

**同场景：走进一个房间**

| 年龄段 | 动作特征 | 写法示例 |
|--------|---------|---------|
| 少年 | 急促、冒失、多余动作多 | 门被一把推开，撞到墙上一声闷响。书袋先甩到榻上，人跟着冲进来，转眼就翻出窗跑了 |
| 中年 | 沉稳、有目的、动作干净 | 推门的动作很轻，进门后先扫视一圈，确认没人后才往里面走。顺手把门带上 |
| 老人 | 缓慢、谨慎、有停顿 | 手搭在门把上停了两秒，缓缓推开一道缝，先探头看了看，确认安全后才慢慢迈进来，一只手扶着门框 |

**方法**：
- 写动作前先确定角色的年龄/身份/性格
- 问自己：这个年龄/性格的人，做这个动作时有什么特征
- 把特征放大1-2个细节写入文中

## 停顿节点法

真正的动作不是连续的，中间有大量的停顿、犹豫、观察。把这些"停顿节点"写出来，动作就立体了。

**常见停顿节点**：
- 伸手触碰某物之前的瞬间犹豫
- 开口说话前的短暂沉默
- 转身离开前的最后一眼
- 接到坏消息时的动作停滞

**写法示例**：
他抬起手，手指悬在门环上方，停住了。（停顿）院内传来练功的破风声。他放下手，从袖中取出传讯玉简看了一眼，又塞了回去。（第二次尝试）再次抬手，这次叩响了门环。

**要点**：
- 停顿节点通常出现在"决定性的瞬间"之前
- 每个停顿节点暗示角色内心在做选择
- 停顿越多，读者越能感受到角色的纠结/紧张
