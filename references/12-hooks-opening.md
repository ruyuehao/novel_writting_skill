# 钩子设计与开篇写法

## 核心理念

在免费阅读平台，读者平均3秒决定是否划走。在付费平台，免费章节的跳出率超过70%。开篇和钩子的目标不是"展现文笔"，而是在最短的文字里让读者产生**不可逆的阅读冲动**。

---

## 一、开篇三要素

### 要素一：冲突前置——让读者停下来

不要告诉读者"将要出事"，让读者亲眼目睹"正在出事"。

**三个标准**：

1. **动作优于描述**
    - 不写"林晚很生气" → 写"林晚掀了桌子"
   - 不写"气氛变得很紧张" → 写"会议室里安静得能听见空调滴水的声音"

2. **对话优于旁白**
   - 不写"他们之间有很深的矛盾"
   - 写"'你凭什么拿我的方案去邀功？'她把文件夹摔在桌上。"

3. **具体优于抽象**
   - 不写"他遇到了大麻烦"
   - 写"讨债的人堵在门口，他摸了摸口袋——身份证和银行卡都在，唯独没有现金。"

**速成模板**：
```
当……的时候，……（突发状况），……（立刻产生的后果）。
```
**示例**：
> 当林晚签完离职协议走出公司大门时，一辆黑色商务车停在他面前。车门打开，里面坐着裁掉他的HR总监和董事长秘书。"林总，董事长想见您。"

### 要素二：人设反差——让读者记住你

人设反差的本质不是"主角有两个身份"，而是两个身份之间存在强烈的**内在张力**。

**三种反差模式**：

**模式一：身份与能力的反差**
弱者身份 + 超常实力。
> 杂役弟子在藏经阁打扫，一群精英弟子闯进来要翻阅禁制内的古籍。杂役弟子头也不抬："禁制是我设的，你们确定要翻？"

**模式二：行为与处境的反差**
极端处境 + 反常行为。
> 宗门演武台上，金丹长老正在训斥一名练气弟子。弟子低头听着，等长老说完，他抬手将擂台地面的青砖按出一个三寸深的掌印："长老，你说的对。但纠正一下，我三天前就已经筑基了。"

**模式三：承诺与实力的反差**
弱者外表 + 强者承诺。
> 退婚当日，对方丢下一句"你配不上我"。主角将婚书收进怀中，平静地说："三年之后，我会亲自登门。届时希望你没有后悔今日说过的话。"

**关键原则**：人设反差要在第一章就建立，不要藏到三五十章才揭秘。

### 要素三：金句钩子——让读者想看下一章

金句钩子不是随便留个悬念，而是一句话让读者产生强烈的情绪反应或认知期待。

**三种最有效的金句钩子**：

**情绪爆发型**——一句话传递强烈情绪，让读者共鸣
> "从今天起，我不做人了。"

**认知颠覆型**——一句话颠覆读者预期
> "其实，你才是克隆体。"

**后果预告型**——一句话预告即将发生的严重后果
> "三秒后，他会死。但他选择了留下。"

**两个操作要点**：
1. 独立成段，位置靠后——钩子放在章节最后一句，单独成段
2. 可截图，能传播——想象读者把这句话截图发朋友圈/抖音，它够不够酷、够不够扎心

### 三要素的组合逻辑

**递进关系**（不是堆砌）：
```
第一段：冲突前置（重点）——让读者停下来
第二段：人设反差（辅助）——让读者记住你
章节结尾：金句钩子（收尾）——让读者想看下一章
```

**常见错误**：三个要素同时出现在第一段，读起来像故事大纲。

## 二、黄金三章

### 第一章：抓住眼球

**字数**：2000-3000字（平台每章标准长度，优先控制在2500字以内）

**目标**：读者点进来之后不想走。

| 位置 | 任务 | 技巧 |
|------|------|------|
| 头500字 | 主角登场 + 冲突前置 | 第一句就要出事，不铺垫不解释 |
| 500-2000字 | 主角首秀，展示性格特质 | 完成第一次人设反差曝光 |
| 2000-2500字 | 章末留钩子 | 悬念型/冲突型/期待型 |

**一句话原则**：第一章的主角必须是一个"有行动的人"，不是一个"被叙述的人"。

### 第二章：立人设 + 亮金手指

**字数**：2000-3000字

| 位置 | 任务 |
|------|------|
| 前半 | 延续第一章冲突或余波，获得转机 |
| 后半 | 初步展示金手指或能力 |
| 章末 | 立下第一个小目标 |

### 第三章：首次冲突

**字数**：2000-3000字

| 位置 | 任务 |
|------|------|
| 前半 | 首次冲突或交锋 |
| 后半 | 让读者看到主角的成长空间 |
| 章末 | 更大的危机或机遇出现 |

**核心**：前三章要完成"立人设 + 展矛盾 + 留悬念"三位一体。

### 开篇检查清单

- [ ] 前300字内有没有发生冲突？读者有没有理由继续看？
- [ ] 第一段没有纯背景介绍（姓名/年龄/家庭/学校等）
- [ ] 主角在前500字内登场了吗？
- [ ] 第一章结尾有钩子吗？
- [ ] 前三章完成了"立人设 + 展矛盾 + 留悬念"？
- [ ] 没有大段世界观说明？
- [ ] 爽点类型与品类匹配（玄幻→实力爽/信息差爽；仙侠→情感爽/反差爽/信息差爽）？

## 三、章首钩子（5种方式）

| 类型 | 写法 | 效果 | 示例 |
|------|------|------|------|
| 冲突切入 | 章首第一句就是危机 | 瞬间拉入紧张状态 | "枪口抵住他的后脑勺。" |
| 悬念前置 | 以未解之谜开头 | 激发好奇心 | "那封信里写的内容，改变了一切。" |
| 信息缺口 | 抛出一个反常信息 | 制造"为什么"的冲动 | "他醒来时发现自己躺在棺材里。" |
| 情绪锚点 | 以强烈情绪收尾的回响开场 | 延续上一章的情绪惯性 | "他说那句话时，眼里的恨意是真的。" |
| 场景快切 | 直接跳到新场景/新时间线 | 避免过渡拖沓 | "三天后，林家大院。" |

**原则**：章首100字内必须让读者知道"这章谁在、在哪、要干嘛"。

---

## 四、章末钩子（5种类型）

章末钩子是追读率的命门。读者决定要不要看下一章，就看最后三行。

| 类型 | 原理 | 示例 |
|------|------|------|
| 悬念式 | 在关键时刻切断，留下未解之谜 | "门外传来一个声音，那是一个不可能出现在这里的人。" |
| 反转式 | 整章暗示A，最后一行发现是B | 整章暗示A是凶手，最后一行的发现B的手上有血 |
| 情绪炸弹式 | 积蓄整章的情绪在章末爆发后切断 | "你从来就没把我当过朋友。"直接切掉 |
| 信息投放式 | 章末抛出一个重磅新信息 | "他翻开日记本的最后一页，上面画着一张和自己一模一样的脸。" |
| 期待式 | 明确预告下一章的重要事件 | "下一章，她终于要说出真相了。" |

**关键原则**：钩子要在写作前就规划好。每章动笔前先想清楚"这章的钩子是什么"，而不是写完了再想怎么结尾。

## 五、爽点在开篇中的应用

> 本节是爽点体系在开篇中的投射，完整手法定义见 `11-story-architecture.md` 爽点章节。

### 开篇最常用的两种爽点手法

**信息差爽（适合玄幻/仙侠）**
> 读者知道主角是转世重生的大能，但宗门弟子全把他当普通杂役。弟子们大谈那位传说中的人物，读者等着看他们知道真相时的表情。

**反差爽（适合玄幻/仙侠）**
> 反派在宗门大殿上当众羞辱主角，全场嗤笑。主角不语，也不辩解。紧接着，一道流光落在大殿之外，守门长老的声音传了进来："山门外有一位前辈持太虚令求见——说是有要事与林小友相商。"

两种手法可以叠加：信息差爽建立"读者知道但角色不知道"的期待；反差爽在真相暴露时释放爽感。

### 章节内爽点→钩子的结构

```
章节前70%：铺垫+推进（建立期待）
章节70%-85%：爽点释放（打脸/反转/收获/升级）
章节最后15%：钩子（抛出新悬念/新危机/新信息）
```

读者满足了（爽点），同时被吊起新的好奇心（钩子），自然要点下一章。爽点与钩子之间不需要过渡，紧挨排布效果最好。

## 六、开篇的品类适配

| 品类 | 开篇偏好 | 钩子风格 | 重点要素 |
|------|---------|---------|---------|
| 玄幻 | 废材逆袭、奇遇觉醒、穿越异界 | 悬念式、反转式 | 信息差爽+实力爽 |
| 仙侠 | 退婚流、宗门试炼、前世今生 | 悬念式、反转式、情绪炸弹式 | 反差爽+信息差爽+情感爽 |

## 七、开篇常见问题与修改

| 问题 | 症状 | 修改方向 |
|------|------|---------|
| 铺垫太长 | 前三章还在介绍背景，没有冲突 | 删掉所有背景介绍，直接从冲突最激烈处切入 |
| 主角太被动 | 主角被事件推着走，没有主动行动 | 让主角在第一章做出一个关键决定或行动 |
| 人设不鲜明 | 读完三章不知道主角是什么性格 | 在前500字内制造一个人设反差点 |
| 钩子缺失 | 章节结尾平缓收拢，读者没有追更冲动 | 每章最后一句单独成段，制造悬念/反转/爆点 |
| 爽点不足 | 读了三章没有一次"爽"的体验 | 检查爽点密度：第一章至少有一个小爽点 |
| 设定轰炸 | 大段说明世界观/力量体系 | 把设定埋进剧情，需要时才透露，不一次性倾倒 |
