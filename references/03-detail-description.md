# 细节描写技巧

## 常见问题

- 人物扁平像纸片，没有辨识度
- 叙述平铺直叙，没有起伏
- 景物道具堆砌，与剧情/人物情绪没关系

## 专属特征设计法

给角色设计2-3个专属特征（小癖好/习惯动作/口头禅），让读者一眼认出。

**示例——林晚的专属特征**：
- 坐下前一定要把桌面上的东西整理整齐
- 把水杯的标签转向正对自己
- 扶眼镜时用中指而不是食指
- 每次翻书前先用拇指捻一下纸边

**设计方法**：
1. 先确定角色的一个核心性格（强迫症/暴躁/温柔/马虎）
2. 从性格推导出日常习惯
3. 选2-3个最具画面感的习惯写进文中
4. 在关键情节中可以打破这个习惯（暗示角色心态变化）

**标准**：读者只要看到这个动作/特征，不需要看名字就知道是哪个角色。

## 小障碍插入法

平铺直叙没有起伏？在角色实现目标的过程中插入2-3个小障碍。

**三类小障碍**：

| 类型 | 示例 | 效果 |
|------|------|------|
| 通话障碍 | 打电话没人接、发消息对方已读不回 | 制造轻微焦虑 |
| 动作打断 | 正要出门发现钥匙不见了、刷卡余额不足 | 制造小挫折 |
| 外因干扰 | 传送阵失灵、下雨没带伞、半路遇到仇家 | 制造意外感 |

**写法对比**：
- 平铺直叙：陈舟早上出门，买完早餐去上班，准时到了公司。
- 插入障碍：陈舟到灵兽铺才发现灵石没带够，翻遍储物袋只有三块下品灵石，刚好够一只最便宜的传讯纸鹤。他付了钱一路小跑到传送阵，阵前排了长队——看了眼时间，他转身奔向城门。

**要点**：
- 障碍要"小"——不改变主线走向，只是增加波折
- 障碍最好和角色性格/当天状态有关
- 每个障碍1-2句话带过，不拖节奏

## 情绪匹配意象法

描写环境/景物时，让景物与角色当下的情绪匹配。

**正向情绪 → 温暖明亮的意象**：
- 阳光、暖茶、微风、鸟鸣、花开
- 光线透过窗帘的柔和、食物的香气
- 热闹但不嘈杂的人群

**负向情绪 → 阴冷压抑的意象**：
- 雨、乌云、寒风、枯叶、影子
- 刺眼的灯光、刺耳的刹车声
- 空荡荡的走廊、冰冷的金属

**反向衬托（高级用法）**：
- 角色刚失恋 → 窗外全是情侣在过情人节
- 角色被人背叛 → 大晴天，阳光刺眼得让人睁不开眼
- 效果：反差越大，情绪越强烈

**要点**：
- 每次场景转换时，先确定角色当下的情绪基调
- 选1-2个与基调匹配的意象自然融入叙述
- 不要为了写景而写景，意象要为情绪服务
